## HSP101 Hexamer structure
Structure (pdb) obtained after MD simulation of an HSP101 hexamer build from aligment of monomers, relaxation and extensive MD equilibration using NAMD software.
We also provide the psf file employed in the NAMD runs.

The electrostatic_potential folder includes calculation of the electrostatic potential of this structure using APBS.
