## Simulations binding of MC1 to HSP101

Resultats de simulacions de dinàmica molecular HSP101 i MC1 amb diferents configuracions inicials:

- A partir de configuració generada per AlphaFold (Joel)
- A partir de configuracions generades per docking utilitzant python i el codi Megadock desde python https://bmcresnotes.biomedcentral.com/articles/10.1186/s13104-023-06505-w

La carpeta comparison conté imatges de les estrcutures finals en cada cas i els pdb

La carpeta simulations conté les trajectòries de les simulacions
